# Top Claude Plugins [![Awesome](https://awesome.re/badge.svg)](https://awesome.re)

A curated list of high-signal Claude plugins, MCP servers, and integrations that meaningfully extend Claude's capabilities in real workflows.

---

## Scope

**Inclusion checklist (meet at least 3 criteria):**

- Extends Claude with a concrete capability (tool use, data access, service integration, workflow automation)
- Actively maintained with evidence of real-world usage (stars, downloads, community activity)
- Built on a standard integration layer (MCP, Claude API tool use, Claude.ai plugin)
- Documented with clear setup instructions and a defined use case

**Exclude or deprioritize:**

- Toy or demo projects without production usage evidence
- Plugins that duplicate a natively supported Claude capability
- Abandoned repos (no activity in 6+ months) unless widely adopted
